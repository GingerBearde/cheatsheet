CSS Display and Positioning Cheatsheet

Display

Property Description Syntax
margin Adjusts the spacing outside of an element margin: [value];
margin: [verticle value] [horizontal value];
margin: [top] [right] [bottom] [left];
padding Adjusts the spacing inside of an element padding: [value];
padding: [verticle value] [horizontal value];
padding: [top] [right] [bottom] [left];
overflow Determines how the browser handles content too large for its container overflow: visible; (takes up extra space)
overflow: hidden; (self-explanatory)
overflow: scroll; (adds an additional scroll bar)
z-index Specifies how far forward or back an element appears on the web page z-index: [value];

Positioning

Property Description Syntax
position Positions elements on the web page. Default value is static position: fixed; (stays in specific spot, IE nav bar)
position: absolute; (positions to closest parent)
position: relative; (positions relative to original spot)
float Determines how far left or right an element should float within its parent element float: left;
float: right;
clear Specifies how an element should behave when it bumps into another element in the same container clear: left;
clear: right;
clear: both;
clear: none;
